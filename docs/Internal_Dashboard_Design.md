## 1. CMS Dashboard Purpose

포트폴리오 콘텐츠(프로젝트, 경력, 스킬 등)를 웹 UI로 관리하기 위한 경량 CMS 대시보드.

| Principle            | Description                                                              |
| :------------------- | :----------------------------------------------------------------------- |
| **Data-First**       | 콘텐츠 관리 효율성 우선                                                  |
| **Minimal**          | 포트폴리오 콘텐츠 CRUD에 집중한 최소 기능                                |
| **SSR-Optional**     | 관리자 전용이므로 CSR SPA로 구현 가능                                    |

## 3. CMS Pages

| Page               | Route Path           | Purpose                                          |
| :----------------- | :------------------- | :----------------------------------------------- |
| **프로젝트 관리**  | `/admin/projects`    | 프로젝트 CRUD (썸네일, 설명, 기술 태그, 링크)    |
| **경력 관리**      | `/admin/experience`  | 경력/학력 항목 CRUD                               |
| **스킬 관리**      | `/admin/skills`      | 기술 스택 CRUD (카테고리, 숙련도)                 |
| **프로필 설정**    | `/admin/profile`     | 개인 정보, 소개문, 프로필 이미지 수정             |
| **문의 내역**      | `/admin/contacts`    | Contact Form 제출 내역 조회 및 관리               |
| **사이트 설정**    | `/admin/settings`    | SEO 메타 정보, SNS 링크, 이력서 파일 관리         |

## 4. CRUD Workflow

| Action    | UI Pattern                              | API Method | 성공 후 동작                   |
| :-------- | :-------------------------------------- | :--------- | :----------------------------- |
| **Create**| "+" 버튼 → Dialog (Form) → 저장         | `POST`     | 토스트 알림 + 테이블 새로고침  |
| **Read**  | DataTable 자동 페이징 + 검색             | `GET`      | -                              |
| **Update**| 행 클릭 또는 편집 아이콘 → Dialog (Form) | `PATCH`    | 토스트 알림 + 해당 행 갱신     |
| **Delete**| 삭제 아이콘 → AlertDialog 확인 → 삭제    | `DELETE`   | 토스트 알림 + 해당 행 제거     |
